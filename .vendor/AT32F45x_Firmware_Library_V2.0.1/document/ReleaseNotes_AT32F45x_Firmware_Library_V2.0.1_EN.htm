Release Notes for

AT32F45x Firmware Library

V2.0.1-2025/11/06

1.      Drivers

l  CRM

w   Modified bit field “adc14rst” to “tmr14rst” and added “adc2en” bit field.

w   Added “crm_interrupt_flag_get()” function.

l  CAN

w   Modified definitions of some register bit fields, and deleted redundant definitions.

l  TMR

w   Updated functions “tmr_pwm_input_config()”, “tmr_encoder_mode_config()” and “tmr_output_channel_config()”. Disable channels before channel mode configuration; otherwise, the channel mode cannot be switched.

2.      Middlewares

l  USB

w   Updated keyboard class, added “send_state” to the “keyboard_type” structure to record the transmission status, and updated function “usb_keyboard_class_send_report” for flow control processing during transmission to avoid data error.

w   Updated “audio class” and modified “audio sampling frequency” request configuration to fix host compatibility issues.

l  EMAC

w   Updated DNS server default IP address to 114.114.114.114.

3.      Demos

l  ADC

w   Updated ADC initialization sequence in all demos to avoid anomalies in extreme situations.

w   Updated all DMA related demos: DMA circular mode is enabled in configuration process, loop trigger is updated and selected for function implementation.

w   Deleted the duplicated “software_trigger_repeat” demo.

l  USB

w   Adjusted the Deepsleep mode entry method (extra low-power mode) and configuration process in keyboard demo, and updated the delay of stabilization after wakeup.

w   Adjusted the Deepsleep mode entry method (extra low-power mode) and configuration process in mouse demo, and updated the delay of stabilization after wakeup.

w   Updated the value of “USB_FLASH_ADDR_OFFSET” in msc from 0x08005000 to 0x08008000 to avoid compilation range overflow in different environments.

w   Updated the function for jumping to APP in the “virtual_msc_iap” demo, and added compilation non-optimization settings to prevent issues where jumps fail due to optimization.

l  Flash

w   Updated “flash_write_read” demo and modified the value of SECTOR_SIZE to 0x800.

l  PWC

w   Updated some demos and deleted the redundant voltage regulation in low-power mode entry process

l  Utilities

w   Updated “qspi_algorithm_demo”, deleted redundant configuration processes, and modified incorrect operation of using null pointers in the code.

l  templates

w   Updated the include path in eclipse_gcc to avoid compilation error caused by workspace_project_locations parameter error while importing multiple projects.

l  Board

w   Revised the writing of some parameter variables in board.c to avoid warning message in the new version of compiler.

l  Cortex-m4

w   Updated the delay judgment in systick_interrupt (modified “>” to be “>=”) to avoid the extra 1ms of IO toggling.

l  AT32IDE and Eclipse projects

w   Deleted “release” option in project configuration, and only the “debug” option is reserved.

l  IAP

w   Updated all IAP_Programmer.exe application programs.

l  Keil project

w   Updated “IROM size” settings in some Keil projects.
