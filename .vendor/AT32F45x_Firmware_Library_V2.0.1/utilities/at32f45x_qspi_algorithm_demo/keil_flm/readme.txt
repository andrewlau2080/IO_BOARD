  this demo is based on the at-start board. qspi flash programming algorithms
  is showed in this demo.  the target flash is esmt32m. 
  when choose qspi1_algorithm demo,the following pin need tobe select
  by Configuration Wizard:
  - QSPI1 CS pin	: PA2/PB6/PB9/PB10/PC11/PG6
  - QSPI1 SCK pin	: PB1/PB2/PB4/PD3/PF10
  - QSPI1 IO0 pin	: PA6/PB0/PB5/PB11/PC9/PD11/PF8/PH2
  - QSPI1 IO1 pin	: PA7/PB7/PB10/PC10/PD12/PF9/PH3
  - QSPI1 IO2 pin	: PA6/PA15/PC4/PC8/PE2/PF7/PG9
  - QSPI1 IO3 pin	: PA1/PB3/PC5/PD13/PF6/PG14
